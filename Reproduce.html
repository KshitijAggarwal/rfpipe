Reproducing Transient Detections¶

The rfpipe library has been developed for the realfast project, a real-time transient search instrument at the VLA. The realfast project will capture VLA visibility data for candidate transients detected in real time. Those results can be reproduced offline by rfpipe.

rfpipe.reproduce.oldcands_read(candsfile, sdmscan=None)[source]¶

Read old-style candfile and create new-style DataFrame Returns a list of tuples (state, dataframe) per scan.

rfpipe.reproduce.oldcands_readone(candsfile, scan)[source]¶

For old-style merged candidate file, create new state and candidate dataframe for a given scan. Requires sdm locally with bdf for given scan.

rfpipe.reproduce.pipeline_candidate(st, candloc, canddata=None)[source]¶

End-to-end pipeline to reproduce candidate plot and calculate features. Can optionally pass in image and corrected data, if available.

rfpipe.reproduce.pipeline_datacorrect(st, candloc, data_prep=None)[source]¶

Prepare and correct for dm and dt sampling of a given candloc Can optionally pass in prepared (flagged, calibrated) data, if available.

rfpipe.reproduce.pipeline_dataprep(st, candloc)[source]¶

Prepare (read, cal, flag) data for a given state and candloc.

rfpipe.reproduce.pipeline_imdata(st, candloc, data_dmdt=None)[source]¶

Generate image and phased visibility data for candloc. Phases to peak pixel in image of candidate. Can optionally pass in corrected data, if available.
